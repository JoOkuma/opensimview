# README #

### FastFuse ###

Fast stack fusion and registration.

### How to build the project? ###

to build the project:

    ./gradlew build

To generate eclipse project files:

    ./gradlew eclipse

To clean-up modified eclipse project files:

    ./gradlew cleanEclipse
